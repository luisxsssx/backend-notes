# BACKEND-NOTES

Notes App Backend

## Tech Stack

**Technologies:** Java, Spring Boot 3, Jason Web Token, Sping Security 6

## Installation

Clone the project and access it

```bash
  git clone https://github.com/luisxsssx/backend-notes.git

  cd backend-notes
```
Install the project

```bash
  mvn spring-boot:build

  mvn spring-boot:run
```
